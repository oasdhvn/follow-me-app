# Follow Me 随行

> 用 AI 记录每一次旅行

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)
[![Next.js](https://img.shields.io/badge/Next.js-15-black)](https://nextjs.org)
[![React](https://img.shields.io/badge/React-18-blue)](https://react.dev)

**Follow Me（随行）** 是一款基于 AI 的旅行攻略卡片生成器。输入目的地，AI 自动为你生成一张包含必游景点、美食推荐、旅行小贴士和行程安排的结构化攻略卡片，支持浏览、分享、点赞和收藏。

---

## 功能特点

- **AI 智能生成** — 输入任意目的地，AI 自动生成完整的旅行攻略卡片
- **精美卡片展示** — 每张卡片采用渐变色设计，包含景点、美食、小贴士和行程
- **互动社区** — 对卡片点赞、收藏、评论，与其他旅行者交流
- **快捷标签** — 预置热门目的地标签，一键快速生成
- **离线感知** — 自动检测网络状态，断网时给出友好提示
- **响应式设计** — 适配桌面端和移动端，支持 PWA
- **防复制保护** — 保护原创内容不被随意复制
